# 💬 Java Chatting Application

## 📌 Project Description
This is a real-time Chatting Application developed using Java. 
The application uses Socket Programming for communication between client and server. 
It allows users to send and receive messages instantly over a network.

## 🛠 Technologies Used
- Java
- Socket Programming
- Swing (GUI)

## ✨ Features
- Real-time messaging
- Client-Server communication
- Simple and user-friendly interface
- Send and receive messages instantly

## 📂 Project Structure
ChattingApplication

## 🚀 How to Run the Project
1. Install Java JDK
2. Open project in NetBeans or any Java IDE
3. Run Server.java first
4. Then run Client.java
5. Start chatting
